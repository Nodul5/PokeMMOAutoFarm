- Installer python

- Installer VSCode Studio ou autre IDE

- Installer tesseract :
https://github.com/UB-Mannheim/tesseract/wiki
-> tesseract-ocr-w64-setup-5.3.1.20230401.exe (64 bit)

- Installer les bibiothèques python avec PIP dans le CMD:
pytesseract, pyautogui, commandes : pip install pytesseract & pip install pyautogui

- Parametrer les variables regionChat, regionDialogue, bindAbra, switchDeuxiemePokemon, (l'endroit où cliquer lors du changement de pokemon lorsque miaouss K.O.) dans functions.py si besoin (celui pour se tp)
Vous pouvez vous aider du test.py si besoin(il vous montrera un screen des régions avec les parametres dans functions.py et des exemples dans imageRegion)

- Sur pokemmo, verrouiller le chat en bas à gauche avec le cadenas, pour qu'il soit opaque, et selectionner l'onglet Battle

- Prendre un abra avec teleport et bind le tp sur 9 (ç).

- Prendre Miaous ramassage avec jackpot comme première attaque et Larcin en deuxieme attaque (Mettre en premier dans l'équipe).

- Aller sur unys dans le pokecentre de Vaguelone.

- Lancer "main.py"
